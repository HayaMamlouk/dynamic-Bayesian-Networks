class src.DynamicBayesNet.DynamicBayesNet(k, separator='#', name_display_mode='compact')[source]

Bases: dCommon

A class to represent and manipulate Dynamic Bayesian Networks (kTBNs). This class extends the functionality of a standard Bayesian Network by incorporating a temporal dimension, allowing the modeling of time-evolving systems.

class src.DynamicBayesNet.dTensor(potential, separator='#', name_display_mode='compact')[source]

Bases: dCommon

A wrapper for a pyAgrum.Potential object that exposes a user-friendly interface using tuple-based variable identifiers (e.g., (‘A’, 1)) instead of the internal string format (e.g., ‘A#1’).

Inherits from dCommon to directly access name conversion functions.

fillWith(value)[source]

Parameters:

  • If a list, it is assumed to be a flattened version of the potential. The length of the list must match the number of cells in the potential.
